## ✨ Overview

**IntelliResult** is a frontend-based Student Performance Analytics Dashboard designed to provide students with an engaging and intuitive way to monitor academic progress.

Instead of displaying plain marks, the system converts academic data into insightful visual analytics, enabling students to better understand their strengths, identify improvement areas, and track overall performance.
